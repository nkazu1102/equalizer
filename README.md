# Original Equalizer

あらゆるウェブサイトの音声に対応する高品質イコライザー拡張機能です。10 バンドの周波数調整が可能で、最高の音質体験を提供します。

## 機能

- 🎚️ **10 バンドイコライザー**: 32Hz から 16kHz まで細かく調整可能
- 🎵 **プリセット機能**: FLAT、PERFECT、ベース強化、トレブル強化、ボーカル強化、ロック、ジャズ、クラシック、エレクトロニックなど
- 💾 **プリセット管理**: カスタムプリセットの保存、読み込み、削除が可能
- 🔄 **リアルタイム調整**: スライダーを動かすと即座に音質が変化
- ⚡ **ON/OFF 切り替え**: ワンクリックでイコライザーを有効/無効化
- 🌐 **全サイト対応**: YouTube Music、YouTube、Netflix、Spotify など、あらゆるウェブサイトで動作
- 🎛️ **自動ゲイン補正**: 音割れを防ぎながら迫力のある音質を実現

## インストール方法

### Chrome/Edge ブラウザ

1. このリポジトリをダウンロードまたはクローンします
2. Chrome/Edge ブラウザで `chrome://extensions/` を開きます
3. 右上の「デベロッパーモード」を有効にします
4. 「パッケージ化されていない拡張機能を読み込む」をクリックします
5. このプロジェクトのフォルダを選択します

### アイコンの作成

`icons` フォルダに以下のサイズのアイコンを配置してください：

- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

アイコンがない場合でも拡張機能は動作しますが、警告が表示される場合があります。

## 使い方

1. 任意のウェブサイト（YouTube Music、YouTube、Netflix、Spotify など）を開きます
2. ブラウザのツールバーにある拡張機能アイコンをクリックします
3. イコライザーパネルが開きます
4. **ACTIVE/INACTIVE**トグルでイコライザーを有効/無効化します
5. プリセットから選択するか、各周波数バンドのスライダーを手動で調整します
6. カスタム設定は「SAVE」ボタンで保存できます
7. 設定は自動的に保存され、次回起動時も維持されます

## 周波数バンド

10 バンドの周波数範囲：

- 32Hz, 64Hz, 125Hz, 250Hz, 500Hz, 1kHz, 2kHz, 4kHz, 8kHz, 16kHz

各バンドは -12dB から +12dB の範囲で調整可能です。

## プリセット

- **FLAT**: すべてのバンドが 0dB（変更なし）
- **PERFECT**: バランスの取れた高品質な音質
- **BASS**: 低音域を強化
- **TREBLE**: 高音域を強化
- **VOCAL**: 中音域を最適化してボーカルを強調
- **ROCK**: ロック音楽に最適化
- **JAZZ**: ジャズ音楽に最適化
- **CLASSICAL**: クラシック音楽に最適化
- **ELECTRONIC**: エレクトロニック音楽に最適化
- **CUSTOM**: 手動調整した設定を保存可能

## 技術仕様

- **Manifest V3**: 最新の Chrome 拡張機能規格に対応
- **Web Audio API**: 高品質なオーディオ処理
- **BiquadFilterNode**: 各周波数帯域の精密な調整
- **自動ゲイン補正**: 音割れを防ぐ高度なアルゴリズム
- **Chrome Storage API**: 設定とプリセットの永続化

## トラブルシューティング

### イコライザーが動作しない場合

1. YouTube Music のページをリロードしてください
2. 拡張機能が有効になっているか確認してください（トグルスイッチ）
3. ブラウザのコンソールでエラーを確認してください（F12 キー）

### 音が途切れる場合

1. 他の拡張機能との競合を確認してください
2. ブラウザを再起動してください
3. 拡張機能を再インストールしてください
